# 制药物料管理法规知识库

## 📖 项目简介

本知识库收录了制药企业药品生产过程中与物料管理相关的法规文件，包括原料、辅料、包材、耗材（硅胶管、取样袋、反应袋、过滤器、层析填料等）的关键条款。

## 📊 知识库统计

| 统计项 | 数量 |
|--------|------|
| 法规总数 | 24 条 |
| 关键条款 | 141 条 |
| 物料管理条款 | ~130 条 |
| 供应商管理条款 | ~11 条 |

## 🔍 功能特性

- **法规索引**：一览所有法规基本信息
- **全文搜索**：按法规名称、关键词快速查找
- **分类筛选**：按物料类型、管理类型筛选
- **原文展示**：关键条款保持原文，不做总结分析
- **一键跳转**：直接访问法规原文网络链接

## 📋 收录范围

### 物料管理
- 接收、存储、领用、使用
- 取样、检测、放行
- 留样、复验、不合格处理

### 供应商管理
- 供应商评估与资质审核
- 审计与批准
- 变更管理
